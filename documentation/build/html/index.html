class noisy_annotation.entity_matching.Entity(entity_id, synonyms, entity_type)¶

A data structure to represent an entity and its metadata.

Parameters

  • entity_id (int) – An identifier for the entity.

  • entity_type (str) – The type of entity.

  • synonyms (list) – A list of synonyms of this entity (the first synonym is the preferred term).

  • end (int) – The ending character of the match.

  • substring (str) – The matched substring.

  • snippet (str) – A snippet showing the match in context.
